# Placement-Portal
The Placement Portal is a web-based platform designed to streamline and digitalize the campus placement process. It helps students, companies, and the placement department manage recruitment activities efficiently.

This system allows students to register, upload resumes, view job opportunities, and apply for jobs. Companies can post job openings and shortlist candidates, while the placement officer (admin) can manage students, companies, job postings, and placement records.
